# Creo Design System

Welcome to the Creo Design System Storybook! This is a comprehensive collection of UI components, patterns, and guidelines for the Creo platform.

## What is Creo?

Creo is an AI-powered stock media and image generation platform that helps creators and businesses find high-quality stock photos, videos, and graphics, plus generate custom images using artificial intelligence.

## Design System Overview

Our design system is built on the principles of:

- **Glassmorphism**: Modern, translucent design with backdrop blur effects
- **Accessibility**: WCAG 2.1 AA compliant components
- **Responsiveness**: Mobile-first design that works on all devices
- **Consistency**: Unified design language across all components
- **Performance**: Optimized for speed and efficiency

## Design Tokens

### Colors
- **Primary**: Orange (#F59E0B) - Brand color for CTAs
- **Secondary**: Purple (#8B5CF6) - Accent color for highlights
- **Success**: Green (#10B981) - Success states
- **Warning**: Yellow (#F59E0B) - Warning states
- **Error**: Red (#EF4444) - Error states
- **Neutral**: Gray scale for text and backgrounds

### Typography
- **Font Family**: Inter (system font fallback)
- **Headings**: Bold, hierarchical sizing
- **Body**: Regular weight, readable line height
- **Code**: Monospace for technical content

### Spacing
- **Base Unit**: 4px (0.25rem)
- **Scale**: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80, 96px
- **Responsive**: Scales with viewport size

### Shadows
- **Small**: Subtle elevation
- **Medium**: Card elevation
- **Large**: Modal elevation
- **Glass**: Backdrop blur effects

## Accessibility

All components are built with accessibility in mind:

- **Keyboard Navigation**: Full keyboard support
- **Screen Readers**: Proper ARIA labels and roles
- **Color Contrast**: WCAG AA compliant ratios
- **Focus Management**: Clear focus indicators
- **Semantic HTML**: Proper HTML structure

- **Bundle Size**: Optimized for minimal impact
